# NetGuard AI
An AI-powered IDS for real-time network monitoring.

## Features
- ML-based anomaly detection.
- Predictive alerts and user feedback retraining.
- Visual cyber battlefield map.

## Installation
1. Clone the repo.
2. Run `pip install -r requirements.txt`.
3. Execute `python netguard.py` (requires admin for sniffing).

## Usage
Monitor your network and receive alerts on threats.
